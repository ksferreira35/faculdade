# Faculdade – Projetos Acadêmicos

Este repositório reúne projetos, exercícios e experimentos desenvolvidos ao longo da minha graduação em Ciência da Computação. O objetivo é registrar minha evolução acadêmica e prática em diferentes áreas da computação.

Aqui estão presentes implementações em linguagens como Java, Python, PHP, HTML, CSS e JavaScript, abordando temas como lógica de programação, desenvolvimento web, criptografia e programação orientada a objetos.

---

## Estrutura do Repositório

```
bootcamp/        Projetos e desafios realizados em bootcamps
cybersecurity/   Estudos e implementações de ciber segurança
dev-interfaces/  Projetos de desenvolvimento de interfaces web
dev-sistemas/    Exercícios e aplicações em PHP
logica-prog/     Exercícios de lógica de programação em Python
POO/             Estudos de Programação Orientada a Objetos
```

* exercícios práticos de algoritmos

